Jasmine matchers that check Angular specific conditions.

检查 Angular 特定条件的 Jasmine 匹配器。

Note: These matchers will only work in a browser environment.

注意：这些匹配器仅在浏览器环境中工作。

例子

期望值是 `Promise`。

期望元素具有给定的文本。
